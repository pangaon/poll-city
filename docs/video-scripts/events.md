# Video Walkthrough: Events
Duration: ~3 minutes
Feature: Events Management

## Script

### Scene 1: Opening (0:00 - 0:10)
VOICE: "Events bring people together. Rally, fundraiser, town hall, canvass launch — Poll City manages the whole lifecycle."

### Scene 2: Creating an Event (0:10 - 0:45)
SCREEN: Click Events in sidebar, then Create Event
VOICE: "Click Create Event. Name it, set the date, time, and location. Add a description. Set capacity if it is limited. Choose visibility — internal for team only, or public for supporters."

### Scene 3: RSVPs (0:45 - 1:15)
VOICE: "Share the event link. People RSVP with their name and email. Poll City tracks going, interested, and waitlisted. If you hit capacity, new RSVPs go to waitlist automatically."

### Scene 4: Check-In (1:15 - 1:40)
VOICE: "At the event, open Poll City on your phone. Tap Check In next to each name as they arrive. You know exactly who showed up."

### Scene 5: Reminders (1:40 - 2:00)
VOICE: "Poll City sends automatic email reminders to RSVPs before the event. No more 'I forgot' — your attendance rate goes up."

### Scene 6: Duplicate and Calendar (2:00 - 2:25)
VOICE: "Duplicate an event for recurring meetings. Export to iCal for volunteers to add to their calendars."

### Scene 7: Closing (2:25 - 3:00)
VOICE: "Host small events. Coffee with the candidate, house party, canvass launch. Every event is a chance to recruit a volunteer and sign up a supporter. After every event, follow up within 24 hours."

## Verification Checklist
- [ ] Event creation works with all fields
- [ ] RSVP form accepts submissions
- [ ] Waitlist triggers at capacity
- [ ] Check-in marks attendance
- [ ] Reminder emails send
- [ ] Duplicate creates copy
- [ ] iCal export downloads valid .ics file
